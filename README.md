# Laminar MVP

Laminar is a dual-mode Zcash transaction-construction toolkit.

- `laminar-core`: business logic for validation, parsing, batching, ZIP-321, UR encoding, QR generation, and receipts.
- `laminar-cli`: dual-mode command-line interface for operator and agent workflows.

## Workspace Layout

- `crates/laminar-core`
- `crates/laminar-cli`
- `desktop` (placeholder for Tauri app, Stage 10)
- `test-vectors`
- `docs`
